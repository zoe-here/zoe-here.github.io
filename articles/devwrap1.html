Wrap GitHub Pages in 10 Minutes

📅 June 18, 2025 🏷️ GitHub, Deployment 🌯 Dev Wrap

For hungry devs in a hurry 🥡 Want a quick and dirty GitHub Pages site? Here's the fastest way to ship it

Step 1: Create Your Repo

Go to GitHub New Repo and:

  • Name it username.github.io
  • Set visibility to Public
  • Check “Add a README” ✅

🧐 Not sure what username means like me?

Click your profile picture (top right) → Your profile →

URL will look like https://github.com/username → That’s your username!

Step 2: Clone It

git clone https://github.com/username/username.github.io.git

Step 3: Add Your First Page

cd username.github.io
echo "Hello World" > index.html

Step 4: Commit & push

git add --all
git commit -m "Initial commit"
git push -u origin main

Step 5: Done? Not Yet...

Oops ❗“Authentication failed”?

Yeah, GitHub disabled password login for Git.

So we need a Personal Access Token (PAT).

But let’s be real... generating a new one each time? No thanks.

🥷 Use SSH Instead (It’s Worth It)

Here’s how to set it up once and forget about it:

📘 Official SSH Guide

🔧 Generate SSH Key

git remote set-url origin git@github.com:username/reponame.git
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
Part Meaning
ssh-keygen Generate SSH key
-t rsa Use RSA algorithm
-b 4096 4096-bit key (stronger than default 2048)
-C Comment (use your GitHub email)

🔑 This gives you two files:

  • id_rsa → your private key (KEEP SECRET)
  • id_rsa.pub → your public key (share with GitHub)

🧷 Add Your Public Key to GitHub

cat ~/.ssh/id_rsa.pub

Copy the output. Then:

  1. Go to GitHub SSH Keys
  2. Click "New SSH Key"
  3. Paste and save

That’s it. No more typing passwords for every push.

🤖 Test Your Connection

ssh -T git@github.com
  • Type yes
  • If it greets you by name → You're in!

Then push again

If it goes through without asking for a password...

🎉 Congratulations! Your GitHub Page is live!

Open: https://username.github.io

Hello World!

Final Tips

  • Don’t share your private key (id_rsa)
  • You can create multiple SSH keys (for personal & work)
  • Accidentally committed your key? Revoke it immediately

Go enjoy your live page!

It may just say "Hello World" today...

But it's a whole playground tomorrow 🌍✨
